This document describes the structure of a portable ProofRail Silver verification package created by the export tool.

## Why source-repo-subset

The signed Silver assertion contains an Ed25519 signature over the raw bytes of the Bronze evidence bundle manifest. The bundle manifest references files using relative paths such as `../../schemas/bronze-claim-schema-v0.1.2.md`.

To preserve the signature, the export tool copies the bundle manifest bytes unchanged and recreates enough of the original repository directory structure so that all relative paths in the manifest resolve correctly within the package.

## Package Manifest

The `package-manifest.yaml` file describes the package contents:

```yaml
package_version: "v0.1.0"
package_type: "proofrail.silver.independent_verification_package"
package_id: "proofrail-silver-demo-002-package"
generated_at: "<ISO-8601 UTC>"
generated_by: "tools/silver/export_independent_verification_package_v0_1_0.py"

source:
  bronze_root: "demos/composed-bronze-demo-001"
  silver_root: "demos/silver-demo-001"

verifier:
  verifier_demo: "silver-demo-002-independent-verifier"
  verifier_version: "v0.1.0"
  expected_report_schema: "silver-verification-report-v0.1.0"

paths:
  signed_assertion: "source-repo-subset/demos/silver-demo-001/runtime/..."
  trust_policy: "source-repo-subset/demos/silver-demo-001/runtime/verifier-b/..."
  revocation_list: "source-repo-subset/demos/silver-demo-001/runtime/verifier-b/..."
  bundle_manifest: "source-repo-subset/demos/composed-bronze-demo-001/..."
  bronze_package_root: "source-repo-subset/demos/composed-bronze-demo-001"
  report_output: "verification-report.json"

limitations:
  - "Local independent verifier demo only."
  - "Not production packaging."
  - "Not third-party certification."
  - "Not Gold certification."
```

## Excluded Files

The export tool excludes:

- `private-key.pem` and `public-key.pem` — Private keys are never packaged.
- `verification-report.json` — Generated fresh by the independent verifier.
- `__pycache__/`, `*.pyc` — Python caches.
- `.DS_Store` — macOS artifacts.
- `*.tgz` — Archives.
- `evidence-freezes/` — Local evidence freeze directories.

## Generating a Package

```bash
python3 tools/silver/export_independent_verification_package_v0_1_0.py \
  --bronze-root demos/composed-bronze-demo-001 \
  --silver-root demos/silver-demo-001 \
  --output demos/silver-demo-002-independent-verifier/runtime/package \
  --force
```

## Reference Tools

The `tools/` directory in the package contains reference copies of the Bronze claim and bundle tooling and the Silver report validator. These are included for auditability and are not used by the independent verifier.
